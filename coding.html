Make a website for free

  1. Go to your app installer, such as microsoft store, App store, etc.
  2. Search Cursor. Install it. Make sure it is related to code.
  3. For now, skip the welcome customizations.
  1. Create a folder in documents naming it anything but make it short.
  2. Open github.com
  3. Create an account.
  4. Under Home it shows start a new repository. In the text box write the name of the repo, don't make it too long.
  5. If you want to show people your website, like my one, click public.
  6. Go to your code editor, preferably cursor. and in the top left you will see a line connecting to three small circles. You can find this under file, edit, selection etc. Click on it.
  7. It is called source control.
  8. Go to the top left again, and click file. Select Open Folder, and open the folder which you created at the start.
  9. On the left of source control it shows a search icon. Left of the search icon it shows a file icon. Click on it.
  10. Right under the file icon is the name of the file. Hover over it. You will see a file option next to the file name and a folder option.
  11. Click the file option. It will create a textbox. Inside the textbox type index.html
  12. Click the file option. It will create a textbox. Inside the textbox type style.css
  13. Click the file option. It will create a textbox. Inside the textbox type script.js

    HTML is the standard language for creating web pages, using tags to define elements like headings, paragraphs, images, and links.

    CSS enhances the appearance of HTML elements by customizing design aspects like colors, fonts, spacing, and positioning.
